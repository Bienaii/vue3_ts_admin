# vue3_ts_admin
It's a vue3+pnpm+vite+ts+elementPlus general project (personal)

## 开发环境
environment  | version
------------- | -------------
node  | v16.15.1
pnpm  | v7.30.0
vite  | v4.3.5
eslint  | v^8.40.0

## 项目运行

```bash
# 使用npm安装pnpm
npm install pnpm@7.30.0 --location=global

# 使用pnpm安装依赖
pnpm install -no-frozen-lockfile --registry=https://registry.npmmirror.com

# 使用pnpm启动服务
pnpm dev
```
